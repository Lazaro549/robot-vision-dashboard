# Cloud AI Robot Vision Dashboard

A simple prototype dashboard for a cloud-ready AI + robotics vision interface.

## Features
- Live dashboard cards for latency, object count, battery, and uptime
- Simulated robot camera view and AI detection summary
- Cloud command queue and architecture notes

## Run locally
From this folder, start a static server:

```bash
python -m http.server 8000
```

Then open http://127.0.0.1:8000/
